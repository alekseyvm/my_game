# Своя Игра - Веб-приложение

Веб-приложение для проведения игры "Своя Игра" в браузере без необходимости установки дополнительных программ.

## Описание

Это полнофункциональное веб-приложение для проведения интеллектуальной игры "Своя Игра" с поддержкой от 2 до 6 игроков. Приложение работает полностью локально в браузере без необходимости подключения к интернету или установки серверного ПО.

## Возможности

- ✅ Поддержка от 2 до 6 игроков
- ✅ Настраиваемые вопросы в формате JSON
- ✅ От 2 до 4 вариантов ответа на вопрос
- ✅ Группировка вопросов по категориям
- ✅ Система начисления баллов
- ✅ Возможность прикрепления изображений к вопросам
- ✅ Дашборд с результатами игры
- ✅ Адаптивный дизайн для разных устройств
- ✅ Сохранение состояния игры в localStorage

## Установка и запуск

### Быстрый запуск (Windows)

1. Скачайте или клонируйте проект
2. Запустите файл **`run.bat`**
3. Приложение автоматически откроется в браузере по адресу `http://localhost:8080`
4. Настройте количество игроков и их имена
5. Нажмите "Начать игру"

Файл `run.bat` автоматически запускает локальный веб-сервер `websitino.exe` на порту 8080 и открывает приложение в браузере.

### Альтернативные способы запуска

Если у вас нет `websitino.exe` или вы используете другую ОС, можно запустить любой локальный веб-сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server

# PHP
php -S localhost:8000
```

После запуска сервера откройте в браузере: `http://localhost:8000`

**Важно:** Для корректной работы загрузки вопросов из JSON файла необходим локальный веб-сервер из-за ограничений CORS в браузерах.

## Формат вопросов

Вопросы хранятся в файле `data/questions.json` в следующем формате:

```json
{
  "categories": [
    {
      "id": 1,
      "name": "Название категории",
      "questions": [
        {
          "id": 1,
          "text": "Текст вопроса",
          "options": ["Вариант 1", "Вариант 2", "Вариант 3", "Вариант 4"],
          "correctAnswer": 0,
          "points": 100,
          "image": "images/question1.jpg"
        }
      ]
    }
  ]
}
```

### Поля вопроса:

- `id` - уникальный идентификатор вопроса
- `text` - текст вопроса
- `options` - массив вариантов ответа (от 2 до 4)
- `correctAnswer` - индекс правильного ответа (начиная с 0)
- `points` - количество баллов за вопрос
- `image` - путь к изображению (опционально)

## Правила игры

1. Игроки по очереди выбирают вопросы из категорий
2. После выбора вопроса отображается карточка с текстом и вариантами ответа
3. Текущий игрок выбирает один из вариантов ответа
4. При правильном ответе игрок получает баллы, при неправильном - теряет
5. Ход переходит к следующему игроку
6. Игра продолжается до тех пор, пока не будут отвечены все вопросы
7. В конце игры можно посмотреть результаты на дашборде

## Добавление своих вопросов

1. Откройте файл `data/questions.json`
2. Добавьте новую категорию или вопросы в существующую
3. Следуйте формату, описанному выше
4. Сохраните файл
5. Перезагрузите игру

## Добавление изображений

1. Поместите изображения в папку `images/`
2. В файле `questions.json` укажите путь к изображению в поле `image`
3. Пример: `"image": "images/my_question.jpg"`

## Решение проблем

### Вопросы не загружаются

Если вопросы не загружаются из JSON файла, это может быть связано с политикой CORS браузера. Решения:

1. Используйте локальный веб-сервер (см. раздел "Установка и запуск")
2. В браузере Chrome можно запустить с флагом `--allow-file-access-from-files` (только для разработки!)
3. Приложение автоматически использует встроенные тестовые вопросы, если не удается загрузить JSON

### Состояние игры не сохраняется

Убедитесь, что в браузере включены cookies и localStorage.
